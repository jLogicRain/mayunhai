# HTML

# 行标签 && 块标签

行标签 (a、span、img)

在一行内显示、不可以进行宽高的数值设定

块标签 (p、div、h1~h6)

独自占领一行、可以进行宽高的数值的设定

嵌套规则

  1. 块标签可以套行标签，行标签不可以套块标签

  2. P标签不要套块属性标签，可以套行标签
